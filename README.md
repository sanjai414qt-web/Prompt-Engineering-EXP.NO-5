# 2.Scenario: Factual Question
**basic Prompt (Input)**

What is photosynthesis?

**basic prompt output**

Photosynthesis is the process by which plants make food using sunlight, water, and carbon dioxide.

**Refined prompt input**

Explain photosynthesis in 100 words, including the process, chemical equation, and its importance to plants.

**Refined prompt output**

Photosynthesis is the process by which green plants produce food using sunlight, carbon dioxide, and water. It occurs in the chloroplasts, where chlorophyll captures light energy. This energy is used to convert carbon dioxide and water into glucose and oxygen.

The chemical equation is:
6CO₂ + 6H₂O + light → C₆H₁₂O₆ + 6O₂

Photosynthesis is essential because it provides energy for plant growth and releases oxygen into the atmosphere. It also forms the base of the food chain, supporting all living organisms directly or indirectly.

# 3. Scenario: Summarization

**Basic prompt**

Summarize blockchain.

**Basic prompt output**

Blockchain is a technology that stores data in blocks linked together. It is secure and used in cryptocurrencies.

**Refined prompt input**

Summarize blockchain technology in 120 words, covering its definition, working principle, and one real-world application.

**Refined prompt output**

Blockchain is a decentralized digital ledger technology used to record transactions securely and transparently. It stores data in blocks, which are linked together in a chain using cryptographic techniques. Each block contains transaction data, a timestamp, and a unique hash of the previous block, making it tamper-resistant.

Blockchain operates without a central authority, relying on a distributed network of computers to validate transactions through consensus mechanisms.

A common real-world application of blockchain is cryptocurrency, such as Bitcoin, where it ensures secure and transparent financial transactions. It is also used in supply chain management, healthcare, and digital identity verification.

# 4. Scenario: Advice / Recommendation

**Basic Prompt input**

How to study better?

**Basic Prompt Output**

You should focus, avoid distractions, and revise regularly. Make a timetable and follow it.

**Refined prompt input**

Provide 5 effective study techniques for high school students, with brief explanations for each.

**Refined Prompt Output**

Active Recall – Test yourself regularly instead of just reading notes. This improves memory retention.

Spaced Repetition – Review topics at increasing intervals to remember them long-term.

Pomodoro Technique – Study for 25 minutes, then take a 5-minute break to maintain focus.

Mind Mapping – Use diagrams to connect ideas, which helps in better understanding.

Practice Questions – Solve past papers to improve accuracy and exam confidence.
